# React HR Portal - Project Writeup

## Overview

This project is a React-based HR Portal created as a course-end practice project. It includes separate dashboard flows for employees and HR users.

## Core Features

- Login for employee and HR user types
- Employee onboarding flow
- Employee self-service view for profile details
- Leave application workflow for employees
- Leave database & status tracking
- HR view for all employee records
- HR approval and rejection of leave requests

## Technical Stack

- React (Vite)
- React Router
- Axios
- JSON Server (mock API backend)
- External component-based CSS

## Styling Approach

The project uses component-based external CSS files along with shared global styles for consistency.

### Responsive Behavior

- Mobile-friendly card and form layouts
- Flexible navigation button wrapping
- Table overflow handling on smaller screens

## AI Usage Statement

GenAI was used to support the production of styling elements in this project and to help draft project documentation.

## Learning Outcomes

- Built multi-route React workflows
- Integrated mock APIs and asynchronous logic
- Applied consistent styling conventions across components
- Improved structure and readability with reusable layout classes

## Status

Completed for educational and portfolio practice.
